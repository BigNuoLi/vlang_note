## strings字符串模块

- new_builder(int) string 

  字符串动态生成器,可以用来,动态生成/拼接字符串内容

- Builder.write(string) 

    写入,不换行,长度变为n

- Builder.writeln(string) 

    写一行,多了一个换行符,长度变为n+1

- Builder.str() string 

    字符串输出

- Builder.cut(int) 

    切掉后几个

- Builder.free() 

    释放字符串内存

- repeat(byte,int) string 

    根据字符以及次数,生成一个字符串
